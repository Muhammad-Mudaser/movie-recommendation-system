Movie Recommendation System

A content-based movie recommendation system built using Python, Pandas, and Scikit-learn.

Features:
- Content-based filtering using cosine similarity
- Recommends top 5 similar movies
- Based on genres, keywords, cast, and director

Technologies Used:
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

How to Use:
1. Download datasets from [TMDB Dataset on Kaggle](https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata).
2. Place `tmdb_5000_movies.csv` and `tmdb_5000_credits.csv` in the project folder.
3. Run the notebook `movie_recommender.ipynb`.
4. Call the `recommend('Movie Name')` function to get top 5 recommendations.
